# React Boilerplate

#### Includes

- webpack
- eslint
- babel
- jest
- less
- webpack-dev-server

#### Installation

```
$ git clone https://github.com/mikeerickson/cd-react-boilerplate
$ npm install 
```

#### Tests

```
$ npm test 

or 

$ gulp mocha
$ jest
```
